Commands:

<argument> is required

[argument] is optional

(click on them to reveal subcommands)

/about

  • /about bot: Information about the bot!
  • /about server: Displays information about the server!
  • /about user <user>: Shows information about a specific user

/fun

  • /fun cats: Shows random pictures of cats!

/moderation

  • /moderation list: Shows list of banned users!
  • /moderation ban <user> [reason]: Bans a user from the server!
  • /moderation clear <amount>: Clears the specified number of messages
  • /moderation kick <user> [reason]: Kicks a user from the server!
  • /moderation mute <user> <time>: Mutes a user for a specified time.
  • /moderation unban <user> [time]: Unbans a specific user!
  • /moderation unmute <user>: Unmutes a user.
