Commands

<arg> is required

[arg] is optional

/about

/about server - Shows information about server

/about user <user> - Shows information about user

/about bot - Shows information about bot

/mod

/mod ban <user> [reason] - Bans a user !

/mod unban <user> [reason] - Unbans a user !

/mod kick <user> [reason] - Kicks a user !

/mod list - Shows a list of banned users !

/mod clear <amount> - Deletes given amount of messages !

/mod mute <user> <time> - Mutes a user !

/mod unmute <user> - Unmutes a user !

/fun

/fun cats - Shows a random cat image.
